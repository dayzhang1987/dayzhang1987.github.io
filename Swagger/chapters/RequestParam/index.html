其前台Ajax请求代码：

   let formData = new FormData(); //创建form对象
    if($("#file")[0].files.length > 0){
        formData.append('file',$("#file")[0].files[0]);//通过append向form对象添加数据
    }
    formData.append('additionalMetadata','11')
    $.ajax({  
        url: 'https://petstore.swagger.io/v2/pet/1/uploadImage', 
        type:'POST',
        data:formData,
        dataType:"application/json", 
        cache: false,
        processData: false,//form对象赋给Ajax的data时阻止它将参数转成成String类型的键值对
        contentType: false,
        success:function(result){ 
            console.log(result)
        }, 
        error:function(result,error){ 
            console.log(result)
        } 
    })

关于跨域访问知识补充：

Access-Control-Allow-Origin：表示允许访问服务器地址来源
Access-Control-Allow-Methods：表示允许客户端访问方法
Access-Control-Allow-Headers：表示允许访问的客户端带的请求头中的参数；
    例如想在请求中添加：codeToken，则其访问的后台必须设置包含codeToken：
    res.addHeader("Access-Control-Allow-Headers", "Origin, X-Requested-With, Content-Type, Accept,codeToken")
Access-Control-Allow-Credentials：表示允许浏览器携带用户身份信息即Cookie
